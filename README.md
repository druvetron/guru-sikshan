## Guru Sikshan - AI-Powered Teacher Training Platform ##
Guru Sikshan revolutionizes teacher professional development by using AI to create personalized, cluster-specific micro-modules based on real-time classroom feedback from teachers and Block Resource Persons (BRPs). Built for scalability from hackathon prototypes to nationwide deployment.

### Features ###
* __AI Personalization__ : NLP and K-Means clustering analyze competency gaps to generate 5-7 micro-lessons (10-15 min each)
* __Mobile-First__ : React Native app with offline access for rural teachers
* __Multi-Role Dashboards__ : Teacher app , principal views , DIET admin analytics
* __Continuous Feedback Loop__ : Used to collect , analyze and improve cycle
* __Scalable Architectur__ : Microservices with Docker, PostgreSQL
